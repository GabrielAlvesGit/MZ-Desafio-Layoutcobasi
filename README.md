# Cobasi - MZ Group 

## 📖 Sobre o Projeto

Este projeto é uma **reprodução educacional** da página inicial da área de **Relações com Investidores** da **Cobasi**, desenvolvida exclusivamente para **fins de aprendizado**. O objetivo é consolidar conhecimentos em **HTML**, **SASS**, **JavaScript**, **JQUERY** e boas práticas de desenvolvimento de interfaces web.

A interface foi recriada com base em treinamentos e práticas, respeitando a identidade visual e a estrutura do layout no **Figma** de Cobasi.  

> **Nota**: Todo o conteúdo, identidade visual e estrutura pertencem à **MZ Group** e **Cobasi**. Este projeto **não é comercial**.
